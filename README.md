```
Private sub command_click
moo = MsgBox("1) Go to www.theprynce.com" & vbCrLf & "2) Vote for it on all the listed TopSites." & vbCrLf & "3) Sit back, relax, and enjoy all the info provided.", vbInformation, "How We Do it!")
end sub
```
